Detailed Description

StateMachine Struct representing a collection of states

  • elevator_cab Elevator_cab object
  • door Door object
  • buttons Allbuttons object of size 4 representing the floors, each floor keeps a set of three buttons
  • stationary Boolean variable that is true when the cab is stationary, and false otherwise
  • stopButton Boolean variable that is true if the stopbutton is pressed, and false otherwise
  • prev_stop_floor integer value representing the last floor the elevator stopped at
