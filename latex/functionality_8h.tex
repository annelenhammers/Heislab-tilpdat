The four if-\/statements are executed sequentially\+:
\begin{DoxyItemize}
\item if someone wants to join the elevator in the current direction at the current floor, the elvator stops
\item if someone wants to get off at the current floor, the elevator stops
\item if none of the above if-\/statements have hits, the elevator scans the button array, looking for orders on the floors above if direction is UP, and below if direction is DOWN, to check if it should go past the current floor
\item if there are no orders above/below, the elevator may stop to pick up passangers going the opposite direction
